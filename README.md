# 📊 Day 3 – Student Marks Analyzer (Python)

This project is a Python-based **Student Marks Analyzer System** that allows users to enter marks for multiple students and calculates important statistics like total, average, highest, and lowest marks.

---

## 📌 Features

* Accepts marks for multiple students
* Uses loop to continuously take input
* Calculates:

  * Total marks
  * Average marks
  * Highest marks
  * Lowest marks
* Handles user decision to continue or stop
* Displays clean result summary

---

## 🧠 Concepts Used

* `while` loop
* Conditional statements (`if`)
* Logical operators (`or`)
* Type conversion (`int`)
* Lists (for iteration)
* Basic data analysis logic

---

## ⚙️ How It Works

1. User is asked whether they want to enter marks
2. If "yes":

   * Enter marks for 5 subjects
   * Program calculates total marks
   * Updates:

     * overall total
     * count
     * highest
     * lowest
3. Repeats until user enters "no"
4. Calculates average
5. Displays final result

---
